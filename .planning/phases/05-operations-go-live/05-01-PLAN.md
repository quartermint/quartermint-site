2. Create `lib/digest/aggregate.ts`:
   - Import `redis` from `@/lib/chat/redis`

   - Import `getISOWeekKey` from `./week-key`
   - Export interface `WeeklyDigestData`:
     ```typescript
     export interface WeeklyDigestData {
       totalSessions: number
       totalMessages: number
       topQuestions: string[]  // max 3
       exportRequests: number
       investViews: number
       weekOf: string  // e.g. "March 27, 2026"
     }
     ```
   - Export `async function aggregateWeeklyStats(): Promise<WeeklyDigestData>`
   - Implementation:
     - Calculate `oneWeekAgo = new Date(Date.now() - 7 * 86400000).toISOString()`
     - Fetch session IDs: `const sessionIds = await redis.lrange('chat:index', 0, 500)` (cap at 500 to avoid timeout; per Pitfall 5 from RESEARCH)
     - For each session ID, fetch `redis.get<ChatLog>(\`chat:${id}\`)`, filter by `log.startedAt >= oneWeekAgo`
     - Count sessions, sum messages.length, extract user-role messages content (first 80 chars), build frequency map
     - Sort frequency map descending, take top 3 as `topQuestions`
     - Read weekly counters: `const weekKey = getISOWeekKey(); const investViews = (await redis.get<number>(\`stats:invest_views:${weekKey}\`)) ?? 0; const exportRequests = (await redis.get<number>(\`stats:export_requests:${weekKey}\`)) ?? 0`
     - Format weekOf: `new Date().toLocaleDateString('en-US', { month: 'long', day: 'numeric', year: 'numeric' })`
     - Return `WeeklyDigestData` object

     // Track /invest page views for weekly digest (OPS-01)
     if (request.nextUrl.pathname === '/invest') {
