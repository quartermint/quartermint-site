Generated by gsd-ui-researcher, verified by gsd-ui-checker. Inherits design system from Phase 1/2/3.

### Upstash Redis Schema

## Smart Starter Chips Spec (ENG-01, D-02)

### Priority Cascade

1. **Scroll context** -- If user has scrolled to a specific section, generate 2 context chips + 1 default
2. **Commit recency** -- If GitHub living signal shows commit within 24h, generate 1 recency chip + 2 defaults
3. **Time-of-day** -- If no scroll or commit context, generate 1 time chip + 2 defaults
4. **Static defaults** -- Fallback when no dynamic context available
