Create `app/api/chat/route.ts` following the AI SDK v6 pattern from RESEARCH.md Pattern 1.

**CRITICAL v6 patterns (do NOT use v3/v4 patterns):**
- Import `streamText`, `UIMessage`, `convertToModelMessages` from `'ai'`
- Import `anthropic` from `'@ai-sdk/anthropic'`
- Import `cookies`, `headers` from `'next/headers'` (MUST await both -- Next.js 16 async APIs)
- Return `result.toUIMessageStreamResponse()` (NOT `toDataStreamResponse()`)
- `convertToModelMessages(messages)` MUST be awaited (async in v6)

**Route structure:**

```
export const maxDuration = 60;

export async function POST(req: Request) {
  1. await cookies() and await headers() -- Next.js 16 async
  2. Extract IP from x-forwarded-for header (split on comma, take first)
  3. Call checkAllRateLimits(ip, cookieStore)
  4. If not allowed: return JSON response with { error: rateCheck.message, type: rateCheck.type } at rateCheck.statusCode
  5. Parse request body: { messages: UIMessage[], sessionId: string }
  6. Validate: messages must be array with length > 0, sessionId must be string
  7. Call streamText({
       model: anthropic('claude-sonnet-4-6'),   // per D-01
       system: buildSystemPrompt(),
       messages: await convertToModelMessages(messages),  // MUST await
       maxOutputTokens: 500,                    // per D-05
       onFinish: async ({ text }) => {
         await logConversation(sessionId, messages, text, ip);
       },
     })
  8. Update cookie: cookieStore.set('qm_chat_count', String(rateCheck.newCount), {
       httpOnly: true, sameSite: 'strict', maxAge: 86400, path: '/', secure: process.env.NODE_ENV === 'production'
     })
  9. Return result.toUIMessageStreamResponse()

  catch: Return 500 JSON with { error: "I'm temporarily offline. You can reach Ryan directly at ryan@quartermint.com.", type: 'error' }
}
```

Do NOT export GET. Only POST is supported.
