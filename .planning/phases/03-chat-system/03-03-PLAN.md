Create `components/chat/chat-mobile-overlay.tsx`:

`'use client'` component.

Props: `{ isOpen: boolean; onClose: () => void }`

**Body scroll lock:**
```typescript
useEffect(() => {
  if (isOpen) {
    document.body.style.overflow = 'hidden';
  } else {
    document.body.style.overflow = '';
  }
  return () => { document.body.style.overflow = ''; };
}, [isOpen]);
```

**Escape key handler:**
```typescript
useEffect(() => {
  if (!isOpen) return;
  const handleKeyDown = (e: KeyboardEvent) => {
    if (e.key === 'Escape') onClose();
  };
  document.addEventListener('keydown', handleKeyDown);
  return () => document.removeEventListener('keydown', handleKeyDown);
}, [isOpen, onClose]);
```

**Focus trap:** On open, focus the close button. Consider using a ref on the overlay container.

**Animation (per UI-SPEC Animation Contract):**
- Entrance: `translateY(100%) -> translateY(0)`, 300ms ease-out
- Exit: `translateY(0) -> translateY(100%)`, 200ms ease-in
- Reduced motion: instant show/hide (no translate, no fade). The global CSS in globals.css handles this via `transition-duration: 0.01ms`.

Implementation: Use state to track animation phase.
- When `isOpen` changes to true: render the overlay, start with `translate-y-full`, then on next frame set to `translate-y-0`
- When `isOpen` changes to false: set to `translate-y-full`, then after 200ms remove from DOM
- Use `transition-transform duration-300 ease-out` for entrance
- Alternatively, use a simpler approach: always render when isOpen, use CSS transition

Simpler approach:
```tsx
if (!isOpen) return null;

return (
  <div
    className="fixed inset-0 z-[60] bg-bg flex flex-col transition-transform duration-300 ease-out"
    role="dialog"
    aria-modal="true"
    aria-label="Chat with Ryan"
  >
    {/* Header: 48px height */}
    <div className="flex items-center justify-end min-h-[48px] px-4 border-b border-text-faint/20">
      <button
        onClick={onClose}
        className="font-body text-[14px] font-semibold text-text min-h-[44px] min-w-[44px] inline-flex items-center justify-center"
        aria-label="Close chat"
      >
        Close
      </button>
    </div>

    {/* Chat content: full height minus header */}
    <div className="flex-1 overflow-hidden p-4">
      <ChatInterface />
    </div>
  </div>
);
```
