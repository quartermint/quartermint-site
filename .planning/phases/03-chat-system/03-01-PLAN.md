**A. Create `lib/chat/rate-limit.ts`:**

Import `Ratelimit` from `@upstash/ratelimit`, `redis` from `./redis`, and types from `./types`.

Create IP limiter:
```typescript
const ipLimiter = new Ratelimit({
  redis,
  limiter: Ratelimit.slidingWindow(60, '1 h'),
  prefix: 'rate',
  analytics: true,
});
```

Export `checkAllRateLimits(ip: string, cookieStore: Awaited<ReturnType<typeof import('next/headers').cookies>>): Promise<RateLimitResult>`:

- Layer 1 (Cookie, per D-08): Read `qm_chat_count` cookie. If `parseInt(value, 10) >= 20`, return `{ allowed: false, message: "Great questions. For a deeper conversation:", statusCode: 429, newCount: currentCount, type: 'rate_limit' }`.
- Layer 2 (IP, per D-08): Call `ipLimiter.limit(ip)`. If `!success`, return `{ allowed: false, message: "You've been busy. Let's continue over a call:", statusCode: 429, newCount: currentCount, type: 'rate_limit' }`.
- Layer 2 catch block (per D-09 fail-closed): On any error, return `{ allowed: false, message: "Chat is temporarily unavailable. Reach out to ryan@quartermint.com.", statusCode: 503, newCount: currentCount, type: 'unavailable' }`. Log error to console.
- Success: return `{ allowed: true, message: '', statusCode: 200, newCount: currentCount + 1, type: 'error' }`.

**B. Create `lib/chat/conversation-log.ts`:**

Import `redis` from `./redis` and types from `./types`.

Export `logConversation(sessionId: string, messages: unknown[], assistantResponse: string, ip: string): Promise<void>`:

- Key: `chat:${sessionId}` (per D-07 prefix).
- Get existing log from Redis. If exists, push new user + assistant messages. If not, create new ChatLog.

- No TTL on stored data (per D-11: kept indefinitely).
